O que é HTML?

HTML, abreviação de Hypertext Markup Language (Linguagem de Marcação de Hipertexto), é uma linguagem de marcação utilizada na construção de páginas na WWW. Criada por Tim Berners-Lee para facilitar a comunição entre usuarios, se formalizou e popularizou-se nos anos 90, recebe atualizações e é mundialmente utilizada até os dias de hoje.

Tags, utilização

As tags HTML são como palavras-chave que definem como o navegador da Web formatará e exibirá o conteúdo. Com a ajuda de tags, um navegador da Web pode distinguir entre um conteúdo HTML e um conteúdo simples. Contêm três partes principais: tag de abertura, conteúdo e tag de fechamento, entretanto algumas tags HTML são tags não fechadas. Quando um navegador de internet lê um documento HTML, o faz de cima para baixo e da esquerda para a direita.

Cada tag HTML tem propriedades diferentes. Um arquivo HTML deve ter algumas tags essenciais para que o navegador da Web possa diferenciar entre um texto simples e um texto HTML. Você pode usar quantas tags quiser de acordo com seu requisito de código.

Atributos

Atributos HTML são palavras especiais que fornecem informações adicionais sobre os elementos ou atributos são o modificador do elemento HTML. Cada elemento ou tag pode ter atributos que irão modificar seu comportamento, inclusive podendo ser aplicados vários atributos numa tag, alterando varias de suas caracteristicas.
